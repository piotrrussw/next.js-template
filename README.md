# Next.js quick start template

### Requirements:

-   node_v12+

### Run:

```bash
npm install
npm run dev
```

### Template

-   Next.js
-   SCSS modules
-   Absolute imports
-   Prop-types
-   normalize.css
-   modern font setup
-   eslint (with react plugin)
-   stylelint
-   prettier
-   import sort
-   tests: jest, testing-library
-   hooks: code validation on: pre-commit
